# Study Tracker (SDV503 – Assignment 2)

This is a simple command‑line Study Tracker built in JavaScript for my SDV503 assignment. The program lets me record study sessions with a topic, number of minutes, and a priority level (High, Medium, Low). I can also list all sessions, view them sorted by priority, and calculate the total minutes I have studied. The goal of this project was to practise input handling, validation, basic logic, and working with arrays in Node.js.

## Features
- Add a study session (topic, minutes, priority)
- Validation for empty topics and invalid minutes
- Rejects invalid priority values
- List all sessions
- List sessions sorted by priority
- Calculate total minutes studied
- Simple menu system using Node’s `readline`
